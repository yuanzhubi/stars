# GitHub Stars Extractor

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
export GH_USER=<username>
export GH_USER=<password>
python stars.py
```
Or, as a one-off:
```bash
GH_USER=<username> GH_PASS=<password> python stars.py
```
